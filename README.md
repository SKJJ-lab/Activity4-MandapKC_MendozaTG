About the Project:
- This project is called Global Explorer. It’s a web app that gets random user data from an API and shows it in a clean and interactive card layout. The goal was to practice fetching data, using it properly, and displaying it in a way that’s easy to understand and visually nice.

What We Implemented:
- Fetching multiple users (you can choose how many, up to 50)
- Displaying each user’s photo, name, email, and country
- Gender filter (Male, Female, All)
- Country filter dropdown
- Sorting users (A–Z, Z–A, Random)
- Flip card effect to show more details (phone, address, age)
- Pin/favorite feature using localStorage
- Loading spinner while data is being fetched
- Error handling using try-catch
- Responsive layout using CSS Grid

Tools Used:
- HTML
- CSS (Grid, Flexbox, animations)
- JavaScript (Fetch API, DOM manipulation)

API Used:
- https://randomuser.me/api/

Notes / Additional Features:
- We added extra features like pinning users, sorting, and animations to make the app more interactive and unique compared to the basic requirements.
